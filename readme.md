### - Résumé du projet
Programme pour savoir les type de bacs collectés au chemin. De plus, possibilité de créer un fichier texte dont l'utilisateur choisi le titre et contenu ainsi que permettre sa lecture.

### - Utilisation
Recherche dans un fichier de type ics, comme un calendrier de la ville. Le but est que l'utilisateur puisse savoir rapidement les prochains bacs qui seront prochainement collectés au chemin. L'algorithme est réfléchi afin que le programme ne lise pas inutilement le restant du fichier, si les deux bacs sont trouvés. De plus, chaque choix est protégé, avec la possibilité d'un retour/continuer entre les boucles de choix. De plus, lorsque necessaire, le programme reconnait si l'entrée de l'utilisateur est un nombre entier ou un caractère.
