### - Résumé du projet
Recherche dans un calendrier de la ville et annonce les bacs qui seront prochainement collectés au chemin. Possibilité de créer un fichier texte dont l'utilisateur choisi le titre et contenu. Par la suite, permet la lecture du fichier texte créer précedemment.

### - Utilisation
Permet , chaque choix est protégé, avec la possibilité d'un retour/continuer dans l'arbre. De plus, lorsque necessaire, le programme reconnait si l'entrée de l'utilisateur est un nombre entier ou un caractère.
